| 阶段 | 周次 | 目标 | 关键动作 | 里程碑 |

| **M1 基建期** | W1-4 | 找手感, 视觉统一, 粉丝 0 → 100 | 拍 8-12 条测试内容,固定模板,头像/简介/封面统一 | 发布 8 条,平均播放 > 200 |

## M1 基建期(Week 1-4)详细

### Week 1: 账号搭建
- [ ] 头像 / 简介 / 背景图统一
- [ ] 固定 1 种视觉模板(色卡 / 字体 / 排版)
- [ ] 自我介绍 1 条(我是谁 / 我做什么 / 关注我能得到什么)
- [ ] 拍 2 条"找手感"内容,不强求数据
- [ ] 注册相关工具:剪映、提词器、数据看板

### Week 2: 试拍 + 复盘
- [ ] 拍 2 条不同角度的同主题内容
- [ ] 对比 2 条数据(完播率 / 互动率)
- [ ] 拆解 5 个同方向 Top 100 爆款
- [ ] 写 1 页"竞品分析"(钩子 / 节奏 / CTA)

### Week 3: 节奏测试
- [ ] 试 3 种发布时段(早 / 午 / 晚)
- [ ] 拍 3 条,1 条/天
- [ ] 测试 3 种不同钩子

### Week 4: 选 1 个方向
- [ ] 复盘 W1-W3 数据,选 1 个"似乎有效"的方向
- [ ] 把这个方向的模板/钩子固定下来
- [ ] 准备 W5 开始高频输出

**W4 末必达**: 8 条发布,平均播放 > 200, 1-2 条 > 500。

## M2 冷启动(Week 5-8)详细

### Week 5-6: 高频 + 钩子迭代
- [ ] 隔日更 1 条
- [ ] 每条都用 W4 选定的模板
- [ ] 钩子微调:数字 / 反差 / 痛点 3 种轮换
- [ ] 每天拆解 1 个新爆款

### Week 7-8: 找爆款公式
- [ ] 把"W4 选定模板 × 当前热点"组合,试 5-8 条
- [ ] 识别第 1 条 > 5000 播放的内容
- [ ] 复盘:这条为什么有效?能复制吗?

**W8 末必达**: 1 条 > 5000 播放,粉丝 > 500,爆款公式雏形。

## M3 第一个爆款(Week 9-12)详细

### Week 9-10: 复制 + 加码
- [ ] 把 W7-8 的爆款公式 1:1 复制 3-5 条
- [ ] 每天看数据,实时调整
- [ ] 钩子、标题、封面分别迭代

### Week 11-12: 卡热点
- [ ] 每天 `/daily` 看热点,优先追热点
- [ ] 24 小时内蹭到的热点流量加成 50%+
- [ ] 视频号 / 公众号同步分发(后期加)

**W12 末必达**: 1 条 > 10w 播放(单平台),粉丝 > 3k。

## M4 稳定输出(Week 13-18)详细

### Week 13-14: 节奏稳定
- [ ] 周更 ≥ 3 条(等效)
- [ ] 建立 30 天内容日历(每月初定)
- [ ] 评论区主动回复(每条 ≥ 5 回复)

### Week 15-16: 互动提升
- [ ] 互动率目标 > 5%
- [ ] 设计 1-2 个"互动型"内容(投票 / 提问 / 翻牌)
- [ ] 开私域引流(主页 / 评论区)

### Week 17-18: 数据复盘
- [ ] W13-18 数据汇总
- [ ] 找 top 3 / bottom 3
- [ ] 调整 W19 后的内容方向

**W18 末必达**: 粉丝 > 1w,互动率 > 5%,稳定周更节奏。

## M5 变现试探(Week 19-22)详细

### Week 19: 准备
- [ ] 检查变现门槛(蒲公英 1w 粉 / 星图 1w 粉)
- [ ] 注册 / 完善商家信息
- [ ] 准备商单介绍卡片

### Week 20-21: 第 1 单 / 第 1 次挂车
- [ ] 主动接洽 3-5 个品牌(蒲公英 / 私域)
- [ ] 或:挂 1 个相关商品,跟踪转化

### Week 22: 复盘变现路径
- [ ] 计算实际 ROI(时间 / 收入)
- [ ] 决定 W23 后主变现方向

**W22 末必达**: 首笔变现 ≥ 100 元,确定主路径。

## M6 放大(Week 23-26)详细

### Week 23-24: 选主平台 + 加大投入
- [ ] 数据复盘:哪个平台 ROI 最高?
- [ ] 该平台产出比例从 50% → 70%
- [ ] 投资 1 个设备升级(麦克风 / 灯光 / 剪辑提速工具)

### Week 25-26: 试第 2 平台
- [ ] 把主平台爆款 1:1 改编,投到第 2 平台
- [ ] 跟踪 2-4 周,看是否值得双线运营
- [ ] 6 个月数据汇总,定下阶段目标

**W26 末必达**: 总粉丝 > 5w,稳定月变现 ≥ 1000 元,有清晰下阶段目标。

## 风险预案

| 风险 | 触发条件 | 应对 |
|---|---|---|
| 平台限流 | 单条播放 < baseline 50% ×3 条 | 暂停发布 3 天,检查内容违规 |
| 内容枯竭 | 1 周找不到 3 个可拍主题 | 回到 `/daily` 强刷热点,加大热点占比 |
| 数据无增长 | M3 末 (W12) 粉丝 < 1k | 切换细分 niche / 加第 2 平台 |
| 失去动力 | 连续 2 周断更 | 降低更新频率到 1 条/周,不放弃节奏 |
| 变现失败 | M5 末 (W22) 0 收入 | 重新评估 niche,考虑换赛道 |
