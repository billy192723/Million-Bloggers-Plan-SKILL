# Instagram 方向(模板)

## Instagram 专属方向

- **主方向**: 
- **Instagram 差异化**: (相比 TikTok,IG 走"美学 + 长尾保存")
- **核心人群**: 25-44 岁, 美 / 欧 / 中东 / 东南亚
- **变现逻辑**: 美学 → 信任 → 品牌 / 私域

## Instagram 爆款公式

### 1. Carousel 教程
"5 [X] tips, swipe to learn"
- 强: 数字 + swipe(多触点)

### 2. Reels 钩子
"This [X] changed my life in [time]"
- 强: 改变 + 时间

### 3. Before/After
"Before: [state A]. After: [state B]. Here's how:"
- 强: 对比 + 教学

### 4. 软推荐
"I've been using [X] for [time]. Here's why I love it"
- 强: 真实 + 体验

### 5. Question 互动
"Comment [X] and I'll send you [Y]"
- 强: 互动 + 价值交换

## 钩子偏好(Instagram Reels 0-3s)

- **前 1 秒必须视觉冲击**
- **on-screen text 是必须的**(静音刷 Reels)
- **9-grid 一致性**: 系列内容用相同色调

## Instagram Reels 30s 标准结构

```
[0-3s]   Hook: visual + on-screen text + voice
[3-10s]  Context: brief setup
[10-25s] Body: 3 quick points, fast cuts
[25-30s] CTA: "Save this" / "Share with a friend"
```

## Carousel 9-grid 设计

- 9 张同色调 / 同字体
- 第 1 张 = 钩子(大字 + 强对比)
- 中间 7 张 = 内容
- 末张 = CTA / 主页 / 关注

## 美学风格(选一种)

- **Minimalist**: 黑白 + 大字 + 留白
- **Bold colors**: 高饱和 + 强对比
- **Soft tones**: 粉 / 米 / 浅蓝
- **Dark mode**: 黑底 + 霓虹色
