# Content Frameworks Reference

Reusable structures for hooks, bodies, CTAs, and platform-specific templates.

## 1. Hook Formulas (First 3 Seconds)

The hook determines if 80%+ of viewers stay. Use these formulas. Always give the user 3 alternatives per card.

### 1.1 反差钩子 (Contrast Hook)
> "5 个工具我砍到 1 个,3 个月后..."

Best for: AI 工具 / 效率 / 极简生活
Hook strength score: 4-5 / 5

### 1.2 数字钩子 (Number Hook)
> "我用 30 天试了 17 个 AI 工具,留下来 3 个"

Best for: 横评 / 教程 / 榜单
Hook strength: 4 / 5

### 1.3 身份钩子 (Identity Hook)
> "作为 XX 3 年的老 XX,说句实话"

Best for: 行业经验 / 专业建议
Hook strength: 3.5-4 / 5

### 1.4 痛点钩子 (Pain Point Hook)
> "你是不是也 XX ?我之前也是,直到..."

Best for: 教程 / 解决方案
Hook strength: 4-5 / 5

### 1.5 预言钩子 (Prediction Hook)
> "2026 年,这 5 个工具会消失"

Best for: 行业评论 / AI / 科技
Hook strength: 4 / 5

### 1.6 反问钩子 (Question Hook)
> "为什么 XX 这么贵还有人买?看完你就懂了"

Best for: 测评 / 揭秘
Hook strength: 3.5 / 5

### 1.7 故事钩子 (Story Hook)
> "3 个月前我还在用 XX,直到那天..."

Best for: Vlog / 个人成长 / 案例
Hook strength: 4 / 5

### 1.8 争议钩子 (Controversy Hook)
> "我用了 5 年 ChatGPT,说实话它不如 Claude"

Best for: AI / 行业评论
Hook strength: 4.5 / 5 (小心平台审核)

### 1.9 悬念钩子 (Mystery Hook)
> "这个工具我不敢公开,但今天必须说"

Best for: 揭秘 / 灰产替代
Hook strength: 4.5 / 5

### 1.10 错误示范钩子 (Mistake Hook)
> "别再 XX 了!90% 的人第一步就错了"

Best for: 教程 / 工具
Hook strength: 4 / 5

## 2. Standard Body Structures

### 2.1 PAS (Problem → Agitation → Solution)
短视频/图文通用,适合教程 / 解决方案类。

```
0-3s:   痛点开场 (你遇到过 XX 吗?)
3-15s:  放大痛点 (我之前每天花 2 小时...)
15-50s: 解决方案 (直到我发现 XX)
50-60s: CTA
```

### 2.2 AIDA (Attention → Interest → Desire → Action)
适合带货 / 种草。

```
0-3s:   Attention (强钩子)
3-10s:  Interest (为什么这个值得看)
10-40s: Desire (展示效果 / 案例)
40-60s: Action (CTA)
```

### 2.3 Hook → Story → Lesson
Vlog / 个人故事 / 经验分享。

```
0-3s:   故事开场
3-40s:  故事展开 (冲突 + 转折)
40-60s: 教训 + CTA
```

### 2.4 数字清单 (Listicle)
横评 / 工具 / 教程。

```
0-3s:   数字钩子
3-50s:  第 1 / 第 2 / 第 3 项(每项 15s)
50-60s: 总结 + CTA
```

### 2.5 Before → After → How
改造 / 教程 / 案例。

```
0-5s:   Before(对比)
5-10s:  After(对比)
10-50s: How(怎么做)
50-60s: CTA
```

### 2.6 对比结构 (Comparison)
适合横评。

```
0-3s:   反差钩子
3-25s:  A 方案 (优劣)
25-50s: B 方案 (优劣)
50-60s: 结论 + CTA
```

## 3. CTA Designs (Call-to-Action)

CTAs that work (vs generic "记得点赞关注"):

### 3.1 评论驱动型
> "你最想看我测哪个?评论区 1 票,票高先做"
> "你买过最坑的 XX 是什么?"

效果: 拉高评论数 + 算法加权

### 3.2 收藏驱动型
> "这张图先收藏,下次 XX 时翻出来"
> "这个清单我整理了完整版,评论区扣 1 我发你"

效果: 拉高收藏数(小红书/Instagram 权重最高)

### 3.3 关注驱动型 (低优先)
> "下期测 X,关注不错过"

效果: 拉高关注转化,但用多了会烦

### 3.4 转化型(带货/私域)
> "链接在评论区第一条"
> "主页有详细教程"

效果: 直接转化,需配合"自然"语境,不能硬广

### 3.5 UGC 触发型
> "你试过 XX 吗?做完 @ 我,挑 3 个展示"

效果: 拉高互动 + 自然涨粉

## 4. Platform-Specific Templates

### 4.1 小红书 Card Template

```
标题: 数字 + 反差 + 痛点 (15-20 字)
封面: 1 大字 + 1 人物/产品 + 1 视觉冲击
开头: 直接抛痛点/反差(不要"今天分享")
正文: 
  - 第 1 段: 钩子 (1-2 句)
  - 第 2 段: 痛点放大
  - 第 3 段: 解决方案 (3-5 个要点)
  - 第 4 段: 案例 / 效果
  - 第 5 段: CTA
标签: 10 个,前 3 个最关键
```

### 4.2 抖音 Script Template

```
[0-3s]  钩子 (必须视觉冲击 + 文字呼应)
[3-7s]  短句交代背景
[7-50s] 主体 (3 个点,每点 14s)
[50-60s] 总结 + CTA
字幕: 全程,大字,关键词高亮
BGM: 平台热门
封面: 标题 + 钩子浓缩
```

### 4.3 B站 Outline Template

```
[0-30s]   钩子 + 视频价值预告
[30-60s]  自我介绍(新号必加,老号可省)
[1-3min]  第 1 点(详细)
[3-6min]  第 2 点(详细)
[6-9min]  第 3 点(详细)
[9-10min] 总结 + 下期预告
字幕: 全程 + 关键帧插图
三连引导: 在 1min、5min、9min 各一次
```

### 4.4 YouTube Outline Template

```
[0-30s]   Hook + What you'll learn + Why trust me
[30-60s]  Context
[1-Xmin]  Body (3-5 sections, each with example)
[X-X+1m]  Recap + CTA (like, sub, comment)
End screen: 20s with 2 videos + 1 sub button
Chapters: Add in description (timestamps)
```

### 4.5 TikTok (海外) Template

```
[0-3s]  Hook (English, direct)
[3-50s]  Body (fast cuts, captions on screen)
[50-60s] CTA in caption (not in video)
Sound:  Trending (check daily)
Text overlay: 大字 (no voiceover needed)
```

### 4.6 公众号 Article Template

```
标题: 32 字以内,带钩子
开头: 100-200 字,讲故事或抛痛点
正文: 3-5 段,每段小标题
结尾: 总结 + 引导"在看"+"关注"
```

### 4.7 X Post Template

```
Line 1: Hook (1 line, ~10 words)
Line 2: 空行
Line 3-5: Story / context
Line 6-10: Lessons
Line 11: Question to invite comments
Hashtags: 3-5
Length: 1200-1800 字符
```

### 4.8 Instagram Carousel Template

```
Slide 1: Hook (大字 + 视觉)
Slide 2-8: 1 point per slide
Slide 9: Recap
Slide 10: CTA + "save this"
Format: 1080x1350 (4:5)
```

## 5. 标题模板库

### 5.1 小红书 标题公式
- 数字 + 痛点: "5 个被高估的 AI 工具"
- 反差 + 数字: "从月薪 5k 到 50k,我做对了 3 件事"
- 教程 + 时间: "3 分钟学会 XX"
- 身份 + 痛点: "作为 5 年 XX 博主,我想说..."
- 警告 + 痛点: "千万别再 XX 了!"

### 5.2 视频号 标题公式
- 简短 + 强情绪: "被裁员后,我做对了这件事"
- 权威 + 数字: "10 年 XX 经验,浓缩成 3 分钟"
- 观点 + 争议: "XX 已经过时,2026 该换 XX"

### 5.3 YouTube 标题公式
- 数字 + 关键词: "5 AI Tools That Replaced My Team in 2026"
- 反差 + 戏剧: "I Tried 30 AI Tools. Here Are the 3 That Won."
- 教程 + 时长: "Learn XX in 12 Minutes"
- 警告 + 关键词: "Stop Using ChatGPT for This (Do This Instead)"

## 6. 视觉规范

### 6.1 封面 (Thumbnail / 封面图)

- 小红书: 1080x1440, 大字 ≤ 10 字, 主色 1 个
- 抖音: 1242x1660 (3:4), 标题 6-12 字
- B站: 1146x717 (16:10), 标题 + 主体清晰
- YouTube: 1280x720 (16:9), 3 元素: 脸 + 文字 + 视觉冲击
- Instagram: 1080x1350 (4:5), Reels 封面同

### 6.2 字幕样式

- 字体: 思源黑体 / 苹方 / Inter
- 大小: 占屏 1/8 高度
- 颜色: 白字 + 黑描边
- 位置: 屏幕中下 (不挡主体)
- 关键词: 黄色或红色高亮

## 7. 30 天内容日历模板

For each niche, here's a standard content calendar mix:

| Week | Mon | Wed | Fri | Sun |
|---|---|---|---|---|
| W1 | 自我介绍/账号定位 | 核心痛点 #1 | 工具/方法 #1 | 行业观点 |
| W2 | 案例 #1 | 核心痛点 #2 | 工具/方法 #2 | 互动型(评论驱动) |
| W3 | 案例 #2 | 核心痛点 #3 | 工具/方法 #3 | 复盘/对比 |
| W4 | 完整教程 | 案例 #3 | 工具横评 | 月度总结 |

Mix: 痛点型 40% + 教程型 30% + 案例/观点型 20% + 互动型 10%

## 8. 频率建议

| Platform | Minimum | Recommended | Burnout Risk |
|---|---|---|---|
| 小红书 | 3 条/周 | 5-7 条/周 | 高频后质量下降 |
| 抖音 | 5 条/周 | 7-14 条/周 | 极易疲劳,需批量生产 |
| B站 | 1 条/周 | 2-3 条/周 | 制作周期长,勿超载 |
| 视频号 | 1 条/周 | 3-5 条/周 | 适合低强度账号 |
| 公众号 | 1 篇/周 | 2-3 篇/周 | 长文写作疲劳 |
| YouTube | 1 条/周 | 2-3 条/周 | 制作周期长 |
| TikTok | 5 条/周 | 7-14 条/周 | 同抖音 |
| Instagram | 3 Reels + 3 Stories/周 | 5 Reels + 7 Stories/周 | Stories 维护成本高 |
| X | 5 条/周 | 7-14 条/周 | 短内容日产 1-2 条,推文串每周 2-3 个 |

If user is solo, **never recommend > 3 platforms simultaneously** for the first 3 months.
