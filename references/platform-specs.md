# Platform Specs Reference

Reference data for 9 platforms the `creator-incubator` skill supports. Use this to make platform-specific recommendations and to calibrate traffic predictions.

## 小红书 (Xiaohongshu / RedNote)

| Field | Value |
|---|---|
| Primary format | 图文 (1/4/6/9 images) + 短视频 (1-3 min) |
| Audience skew | 70% female, 25-35 main, tier-1-2 cities |
| Best length | 图文 600-1200 字, 短视频 60-90 秒 |
| Peak publish times | 12:00-13:00 (午休), 19:00-22:00 (晚高峰) |
| Peak engagement days | 周二-周五, 周日晚 22:00 后是低峰 |
| Tag limit | 10 tags recommended, 20 max |
| Algorithm weights (estimated) | 完读/完播 30%, 互动率 25%, 搜索命中率 20%, 账号权重 15%, 实时热度 10% |
| Avg engagement rate (top 10%) | 6-12% |
| Avg engagement rate (median) | 2-4% |
| Avg views (新号前 10 条) | 200-2000 |
| Avg views (1w 粉账号) | 3000-15000 |
| Search SEO value | 极高 — 笔记半年后仍能长尾 |
| Monetization | 蒲公英 (品牌广告, 1w 粉门槛), 薯条 (付费推广), 带货, 私域引流 |
| Pitfalls | 标题党 → 限流, 引流微信 → 限流, 同质化 → 限流 |

**Niches that work**: 美妆, 穿搭, 家居, 母婴, 美食, 旅行, 健身, 学习, 工具评测, 自我提升, 副业, AI 工具, 摄影.

**Niches that struggle**: 极客硬核技术, 财经深度, B2B, 男士穿搭(天花板低).

## 抖音 (Douyin)

| Field | Value |
|---|---|

| Audience skew | 60% male-leaning on tech/finance; 60% female on lifestyle |
| Best length | 21-34 秒 (sweet spot for completion rate) |
| Peak publish times | 12:00-14:00, 19:00-22:00 |
| Peak engagement days | 周五-周日 |
| Hashtag | 3-5 个, 不超过 8 个 |
| Algorithm weights | 完播率 35%, 互动率 25%, 关注转化 15%, 转发 10%, 评论质量 10%, 账号权重 5% |
| Avg engagement rate (top 10%) | 5-10% |
| Avg engagement rate (median) | 1-3% |
| Avg views (新号前 10 条) | 500-5000 |
| Avg views (10w 粉账号) | 10w-100w |
| Search SEO value | 中等 — 7 天热度衰减明显 |
| Monetization | 星图 (1w 粉门槛), 直播带货, 团购, 团购达人, 橱窗 |

## B站 (Bilibili)

| Field | Value |
|---|---|
| Primary format | 5-20 分钟横屏长视频 (16:9) |
| Audience skew | 65% male, 18-30 main, 强学生/年轻上班族 |
| Best length | 8-12 分钟 (中视频计划区间) |
| Peak publish times | 17:00-20:00 工作日, 14:00-18:00 周末 |
| Peak engagement days | 周五-周日 |
| Tag | 5-10 个, 垂直优先 |
| Algorithm weights | 三连率 (点赞+投币+收藏) 30%, 完播率 25%, 评论率 15%, 弹幕率 10%, 完播时长 10%, 账号权重 10% |
| Avg engagement rate (top 10%) | 8-15% (B站互动率含弹幕) |
| Avg engagement rate (median) | 3-6% |
| Avg views (新号前 10 条) | 100-1000 |
| Avg views (1w 粉账号) | 5000-50000 |
| Search SEO value | 高 — 视频长尾 6-12 个月 |
| Monetization | 创作激励 (1000 粉, 低单价), 充电 (粉丝订阅), 直播, 带货, 商单 |

## X (Twitter)

| Field | Value |
|---|---|
| Primary format | 短文本(≤280 字符 免费 / 25000 Premium)+ 图片(最多 4 张)+ 短视频(≤140s)+ 推文串(1/n) |
| Audience skew | 25-44 偏多, 强新闻/科技/创业, 全球(英语圈为主) |
| Best length | 单条 ≤140 字符 (高完播); 推文串 7-15 条,每条独立成立 |
| Peak publish times | 美东 8-10am / 12-1pm / 5-6pm = 北京 21-23 / 0-1 / 5-6 |
| Peak engagement days | 周二-周四(英文圈),周末偏低 |
| Tag | 最多 2-3 个 hashtag,优先短词(`#AI` `#buildinpublic` `#startup`) |
| Algorithm weights (estimated) | 互动率(点赞+RT+回复)30%, 转发 25%, 点击 20%, 停留时长 15%, 关注转化 10% |
| Avg engagement rate (top 10%) | 3-8% |
| Avg engagement rate (median) | 0.5-2% |
| Avg impressions (新号前 30 条) | 50-500 |
| Avg impressions (1k 粉账号) | 5000-30000 |
| Search SEO value | 高 — 推文永久可搜(蓝标),新闻类长尾流量稳定 |
| Monetization | X Premium 订阅分成(500 万展示/月), Tips 打赏, X Subscriptions(100 粉), Community 收费群, 站外引流 |
| Pitfalls | 280 字上限前 80 字最关键; 推文串每条必须独立成立; 链接推文算法降权(放评论区); 中文生态比英文弱 |

**X 特有机制**:
- **推文串(Thread)**:长内容拆 1/ 2/ 3 ... ,首条强钩子,后续每条独立
- **Quote Tweet(引用转发)**:转发 + 加评论,比纯 RT 多 2-3 倍曝光
- **Community**:建自己的社群(类似微信群),推主时间线
- **Spaces**:语音直播,蹭热度上推荐位
- **Bookmarks**:"被收藏"是权重信号,写"值得收藏"型推文
- **Premium 长文**:Premium+ 可发 25000 字长文 + 收入分成

**Niches that work**: 创业(build in public), 科技/AI, 营销, 加密货币, 新闻评论, 自我提升, 设计, 开发者文化, 设计, 商业评论.

**Niches that struggle**: 纯视觉(Instagram 更适合), 长教程(B站/YouTube 更好), 中文泛娱乐(用户群不匹配), 美妆/时尚(更适合 IG).
